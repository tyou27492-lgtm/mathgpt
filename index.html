About MathGPT Assistant

MathGPT Assistant is an advanced AI-powered mathematical problem solver that provides step-by-step solutions to a wide range of mathematical problems.

Features:

  • Step-by-step math problem solving
  • Multilingual support (including Bangla)
  • Chat history persistence
  • Custom math keyboard
  • Mobile-responsive design

Help & Usage

How to Use:

  1. আপনার গণিত প্রশ্নটি ইনপুট বক্সে লিখুন / Type your math question in the input box
  2. Enter চাপুন বা Send বোতামে ক্লিক করুন / Press Enter or click Send button
  3. ম্যাথ কীবোর্ড টগল করুন সহজে প্রতীক ইনপুটের জন্য / Toggle math keyboard for easy symbol input
  4. বাম দিকের প্যানেলে আপনার চ্যাট ইতিহাস দেখুন / View your chat history in the left panel

Supported Topics:

  • পাটিগণিত এবং মৌলিক ক্রিয়াকলাপ / Arithmetic and basic operations
  • বীজগণিত (সমীকরণ, অসমতা) / Algebra (equations, inequalities)
  • জ্যামিতি এবং ত্রিকোণমিতি / Geometry and trigonometry
  • ক্যালকুলাস (ডেরিভেটিভ, ইন্টিগ্রাল) / Calculus (derivatives, integrals)
  • পরিসংখ্যান এবং সম্ভাব্যতা / Statistics and probability
  • পিথাগোরাস উপপাদ্য / Pythagorean theorem
  • এবং আরও অনেক কিছু! / And much more!

Tips:

  • আপনার প্রশ্নগুলি সুনির্দিষ্ট করুন / Be specific with your questions
  • সমস্ত প্রাসঙ্গিক তথ্য অন্তর্ভুক্ত করুন / Include all relevant information
  • সঠিক গাণিতিক স্বরলিপি ব্যবহার করুন / Use proper mathematical notation
